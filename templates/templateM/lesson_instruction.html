Управление банком уроков

Урок - объект, позволяющий настроить занятие с сообщениями из БД

Основные характеристики

  • Вопросы - сообщения из БД
  • Время - ограничение урока по времени(0 - нет ограничения)
  • Цена за ответы - опыт за (не)правильные ответы

"Создание"

1) Придумаете уникальное название урока

2) Укажите время, если хотите его ограничить. Иначе пропустите

3) Укажите цену за ответы. В режиме тестирования будет использоваться именно она

(При создании сообщений указывается цена для режима тренировки)

4) Выберете сообщения для урока или укажите количество (они выберутся автоматически)

"Редактирование"

Вы можете:

  • Посмотреть список созданных уроков
  • Отредактировать характеристики как при создании
  • Удалить уроки
