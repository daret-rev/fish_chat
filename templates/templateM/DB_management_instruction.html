Управление банком заданий

Данная панель упраления позволит вам управлять БД с заданиями без СУБД и специальных навыков

"Создать"

Введите текст сообщения и коментарии

Выберете правильный ответ и настройте опыт

После сохранения задание будет автоматически добавленно в БД и готово к использованию

"Изменить"

Этот режим позволяет управлять уже созданными сообщениями в БД

"Удалить"

Безвозвратно удаляет сообщение из БД - будьте осторожны

"Редактировать"

Позволяет изменять сообщения из БД, настраивая их как при создании

Фильтрация

Вы можете использовать удобный поиск сообщений в БД:

  • По их ID из БД
  • По ключевым словам в сообщении (регистр не важен)
  • По ответу (REAL/FAKE)
