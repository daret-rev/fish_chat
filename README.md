# Project Fish-Chat

![Data Illustration](image/logo.png)

## Описание
Проект создан для интерактивного обучения распознования распознованию мошенничества. Его цель наглядно продемонстрировать возможные мошеннические уловки, используемые в СМС-сообщениях, писмах и телефонных звонках.

Вы сможете
============
   - Проверить свои навыки и внимательность
   - Узнать о возможных угрозах и научиться их выявлять
   - Получить помощь в анализе сообщений от бота-помошника
## Установка
```sh
git clone https://github.com/a20atamcom-beep/fish_chat.git
```
Для работы некоторых функций необходима локальная версия модели. Получить её можно здесь:
https://lmstudio.ai/download
Видимость модели определяется в docker-compose.yml
```
 BASE_URL: http://192.168.3.8:1234/v1 
```
## Запуск
```sh
docker compose up -d --build
```
